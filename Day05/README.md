## Overview
Day 5 focused on combining **DSA array logic** with **React conditional rendering**.  
Built an interactive **Array Tool** to reverse, rotate, and show/hide arrays dynamically.

---

## Features
- Reverse array using `.reverse()`
- Rotate array to the right using `.slice()` and spread operator
- Show/hide array with React conditional rendering (`&&`)
- Interactive UI with buttons

---

## Key Takeaways
- **DSA + React** works together: array manipulations update UI via state  
- Use **keys** when mapping arrays for proper rendering  
- Conditional rendering enables dynamic and interactive interfaces
